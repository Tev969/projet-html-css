Les concepts du développement web pour tous?

Les objectifs ?

Les objectifs sont de vous faire découvrir de manière simple le concept du dévellopement web ainsi que les technologies utilisé ! De sorte a connaitre mieux et plus spécifiquement l'intêret de chaque outils ainsi que directement du métier en lui même.

Le dévellopement web ?

Son but est de développer des interfaces web adaptées aux besoins de son client. Il peut s'agir de sites Intranet, de sites internet, d'applications web ou d'outils en ligne. Il répond à un cahier des charges élaboré par le client ou un chef de projet.

Les différentes technologies présenté

Nous aborderont les technologies sous forment d'image mental ! Tout d'abbord le HTML représentant le squelette d'un site , Le CSS quand à lui servire à styliser notre squelette , ensuite le Java Script ou J.S , servant a réprésenter les organes d'un site.
